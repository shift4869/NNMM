# NNMM


## 概要
NNMM = NicoNicoMylistManager  
特定のマイリストや特定の投稿者の投稿動画を一覧で管理するマネージャ


## 特徴（できること）
- 実装予定機能
    - 特定のユーザーの投稿動画を一覧表示
    - 特定のマイリスト一覧表示
    - 新着強調
    - 動画情報表示


## 前提として必要なもの
- Pythonの実行環境(3.9以上)
- ニコニコのアカウント情報


## 使い方
1. 後で追記
